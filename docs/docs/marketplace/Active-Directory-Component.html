Login - Active Directory Component

Overview

The Login - Active Directory component provides a front-end user interface that contains a text fields for the Username and Password, and a button to Sign In. The component links to a back-end service that provides authentication using Microsoft Active Directory.

You can use this component in any scenario where you want to add authentication to your app. You can modify the placeholders for the text fields, and modify the look and feel using skins. You can also map the component to your own Active Directory service.
