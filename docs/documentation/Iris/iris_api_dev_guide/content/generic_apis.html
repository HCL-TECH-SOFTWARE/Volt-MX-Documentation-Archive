Standard HCL API

The Standard HCL API provides a set of functions as part of the hcl Namespace that can retrieve and modify data from files, print data in the log files, and run tasks on worker threads. The Standard VoltMX Iris API also contains functions and constants of the hcl.nosql API to connect to a database and modify the data.

The Standard VoltMX Iris APIs use the hcl Namespace and the following API elements.

VoltMX Iris APIs

Function Description
[hcl.convertToBase64](hcl_functions.htm#convertt) Converts rawbytes (returned by the camera or the encryption API) to a base64 encoded string.
[hcl.convertToRawBytes](hcl_functions.htm#convertr) Provides your app with the ability to read rawbytes from a base 64 encoded string.
[hcl.evaluateJavaScriptInNativeContext](hcl_functions.htm#evaluateJavaScript) Enables a web app JavaScript module, which is running in the Browser widget, to execute JavaScript code in the VoltMX Iris native context.
[hcl.getError](hcl_functions.htm#getError) Retrieves an error object from a handle to the error object.
[hcl.props.getProperty](hcl_functions.htm#props.ge) Used to access parameters from an external properties file.
[hcl.print](hcl_functions.htm#print) Prints debugging output.
[hcl.runOnMainThread](hcl_functions.htm#hcl.run) Helps you run the JavaScript code on the main thread. It is an asynchronous API.
[hcl.runOnWorkerThread](hcl_functions.htm#hcl.run2) Provides apps with multithreding capabilities.
[hcl.type](hcl_functions.htm#type) Retrieves the data type of the specified input.

 

Convert rawbytes returned by the camera or encryption APIs to a base64 encoded string using the [hcl.convertToBase64](hcl_functions.htm#convertt) function. To provide your app with the ability to read rawbytes, use the [hcl.convertToRawBytes](hcl_functions.htm#convertr) function. Use the [hcl.getError](hcl_functions.htm#getError) function to retrieve an error object from the try/catch blocks. Using the [hcl.props.getProperty](hcl_functions.htm#props.ge) function, you can access parameters from an external properties file. Print the debugging output using the [hcl.print](hcl_functions.htm#print) function, and retrieve data types of inputs using the [hcl.type](hcl_functions.htm#type) function. To run a JavaScript code on the Main Thread, use the [hcl.runOnMainThread](hcl_functions.htm#hcl.run) function. To provide multithreading capabilities to an app, use the [hcl.runOnWorkerThread](hcl_functions.htm#hcl.run2) function.

hcl.nosql APIs

Function Description
[hcl.nosql.addRecords](hcl_functions.htm#hcl.nosql.addRecords) Replaces a record if the primary key matches, else the record is added.
[hcl.nosql.addOrReplaceRecords](hcl_functions.htm#hcl.nosql.addOrReplaceRecords) Replaces a record if the primary key matches, else the record is added.
[hcl.nosql.clearTable](hcl_functions.htm#hcl.nosql.clearTable) Clears the existing data in a table in the callback of openTransaction API.
[hcl.nosql.closeDatabase](hcl_functions.htm#hcl.nosql.closeDatabase) Closes the database connection.
[hcl.nosql.createIndices](hcl_functions.htm#createIndices) Creates indices for the specified table in the database.
[hcl.nosql.createTable](hcl_functions.htm#hcl.nosql.createTable) Creates a table (object store) in the indexed database.
[hcl.nosql.databaseExists](hcl_functions.htm#hcl.nosql.databaseExists) Specifies whether the required database exists or not.
[hcl.nosql.deleteDatabase](hcl_functions.htm#hcl.nosql.deleteDatabase) Deletes an existing database.
[hcl.nosql.deleteIndices](hcl_functions.htm#deleteIndices) Deletes indices for the specified table in the database.
[hcl.nosql.deleteRecords](hcl_functions.htm#hcl.nosql.deleteRecords) Deletes the rows that match the specified condition.
[hcl.nosql.deleteTable](hcl_functions.htm#hcl.nosql.deleteTable) Deletes a table in the database.
[hcl.nosql.fetchRecords](hcl_functions.htm#hcl.nosql.fetchRecords) Reads the rows of a table.
[hcl.nosql.getPrimaryKeys](hcl_functions.htm#hcl.nosql.getPrimaryKeys) Fetches the primary key of the required table.
[hcl.nosql.getTables](hcl_functions.htm#hcl.nosql.getTables) Returns the list of tables in the selected database.
[hcl.nosql.openDatabase](hcl_functions.htm#hcl.nosql.openDatabase) Opens an existing database; if the database does not exist, however, the API creates the database and then opens it.
[hcl.nosql.openTransaction](hcl_functions.htm#hcl.nosql.openTransaction) Opens a transaction in which you can execute any of the insert, update, or delete operations.
[hcl.nosql.replaceRecords](hcl_functions.htm#hcl.nosql.replaceRecords) Replaces the records that match the provided condition. The API also updates the full record with a new set of columns.
[hcl.nosql.tableExists](hcl_functions.htm#hcl.nosql.tableExists) Returns the appropriate Boolean value depending if the required table exists or not.
[hcl.nosql.updateRecords](hcl_functions.htm#hcl.nosql.updateRecords) Updates rows and specific columns that match the provided condition.

 

Further, you can check whether a required database exists using the [hcl.nosql.databaseExists](hcl_functions.htm#hcl.nosql.databaseExists) function. Use the [hcl.nosql.openDatabase](hcl_functions.htm#hcl.nosql.openDatabase) function to open an existing database, or create a new database and open it. Once the database is open, use the [hcl.nosql.openTransaction](hcl_functions.htm#hcl.nosql.openTransaction) function to open a transaction and execute operations. To close the connection to a database, use the [hcl.nosql.closeDatabase](hcl_functions.htm#hcl.nosql.closeDatabase) function. Delete existing databases using the [hcl.nosql.deleteDatabase](hcl_functions.htm#hcl.nosql.deleteDatabase) function.

To check whether a table exists in the database, use the [hcl.nosql.tableExists](hcl_functions.htm#hcl.nosql.tableExists) function. Use the hcl.nosql.createTable function to create a table in an indexed database. Retrieve the primary key of a table using the [hcl.nosql.getPrimaryKeys](hcl_functions.htm#hcl.nosql.getPrimaryKeys) function. To retrieve a list of tables present in the database, use the [hcl.nosql.getTables](hcl_functions.htm#hcl.nosql.getTables) function. To delete existing data in a table, use the [hcl.nosql.clearTable](hcl_functions.htm#hcl.nosql.clearTable) function, and delete a table in the database using the [hcl.nosql.deleteTable](hcl_functions.htm#hcl.nosql.deleteTable) function.

Add a record in a table using the [hcl.nosql.addRecords](hcl_functions.htm#hcl.nosql.addRecords) function. To replace records matching a specific condition, use the [hcl.nosql.replaceRecords](hcl_functions.htm#hcl.nosql.replaceRecords) function. Replace a record matching the primary key using the [hcl.nosql.addOrReplaceRecords](hcl_functions.htm#hcl.nosql.addOrReplaceRecords) function. To view the rows of a table, use the [hcl.nosql.fetchRecords](hcl_functions.htm#hcl.nosql.fetchRecords) function. Update the rows and columns of a table using the [hcl.nosql.updateRecords](hcl_functions.htm#hcl.nosql.updateRecords) function. Use the [hcl.nosql.deleteRecords](hcl_functions.htm#hcl.nosql.deleteRecords) function to delete the rows matching a specific condition.

Note: The setAppHeaders, setAppFooters, appreset, readfrombase64 APIs have been deprecated and must not be used to develop new software. However, documentation for them is provided to help in the maintenance of legacy software.
