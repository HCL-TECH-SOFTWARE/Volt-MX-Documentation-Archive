Android Quick Start Guide

Introduction

Follow the instructions in this topic to create a simple Iris application that uses the Native Function APIs for Android.

Prerequisites

In order to use this guide, you will need:

  • a computer running a recent version of Windows or Mac OS.
  • [VoltMX Iris Classic (formerly Iris Enterprise)Version 7.2](http://docs.hcl.com/hcllibrary/iris/iris_user_guide/Default.htm#SUGinstallHCLStudio.htm?TocPath=Configuring%2520Your%2520Computer _____2) or later.
  • The Android SDK and tools, as well as a configured emulator or device. The VoltMX Iris Preview app will not run this application.
  • Some knowledge of Android development.
