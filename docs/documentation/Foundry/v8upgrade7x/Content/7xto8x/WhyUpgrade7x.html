Key Features in V8

MADP

The following table details the pre-release overview of HCL Iris and HCL Foundry.

HCL Iris: HCL Iris is an integrated, intuitive development environment to build native, HTML5, and hybrid apps across phones, tablets, wearables, and desktops – all from a single code base.

The release overview features listed in the following table are from the release version 2.5 (August 2015) to release version V8 (April 2017).

For more information on HCL Iris features, refer HCL Iris Release Notes.

<p>Iris V8</p>
