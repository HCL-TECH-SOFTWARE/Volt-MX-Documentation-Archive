Configure Load Balancer as Proxy in JBoss Multinode Setup (Optional if SSL Offloading at Load Balancer)

  1. Navigate to your VoltMX Foundry install directory and open the domain.xml file.

  2. In the domain.xml, update all the http connector tags to include the scheme and proxy-port details.

    For example, `<connector name="http" protocol="HTTP/1.1" scheme="https" socket-binding="http" proxy-name="loadbalancer hostname" proxy-port="443" secure="true"/>`
