Installing HCL Foundry

Note: If a user installs HCL Foundry Console on a partition different from the user’s home directory, the user must have read/write execute permissions.

This section of the document provides you with the instructions for installing and configuring HCL Foundry Console.

Important: Ensure that you have the required hardware and access to the supporting software as mentioned in the Prerequisites section and performed the steps as mentioned in the Pre-installation Tasks section.

Make sure you have proper administrative rights to install HCL Foundry on your computer.

You also need a user where HCL Foundry will be installed.

The **** directory contains the log files documenting each invocation of the installer. To make problem identification easier, provide these log files to HCL when reporting an issue.

Single-Node Installation Procedure

To install HCL Foundry Console using the installer, follow these steps:

  1. Execute the following command to switch to the user that you created.

# su - username

  1. Press Enter. You will be prompted to enter the password.
  2. Enter the password, and press Enter.
  3. Unzip the downloaded zip file, and go to the folder HCLFoundry_Installer_Linux.
  4. Start the ./HCLFoundryConsoleInstaller-6.0.1.GA.bin file by executing the following command:
